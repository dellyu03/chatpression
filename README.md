# ChatPression

## About the project

Chatpression is an AI chatbot website that identifies your conversation personality type and provides conversation solutions tailored to your personality type.

## Key Differentiator

Chatpression's key differentiator is that it analyzes conversation types not through rigid questionnaires, but through the history of conversations with character chatbots.
Through the onboarding process, you can chat with an AI chatbot tailored to your situation.
