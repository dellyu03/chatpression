# ChatPression

## About the project

Chatpression is an AI chatbot website that identifies your conversation personality type and provides conversation solutions tailored to your personality type.

## Key Differentiator

Chatpression's key differentiator is that it analyzes conversation types not through rigid questionnaires, but through the history of conversations with character chatbots.
Through the onboarding process, you can chat with an AI chatbot tailored to your situation.
Through the onboarding process, ChatPression creates a character chatbot tailored to the user's information, enabling a natural first-meeting conversation experience.
This allows users to learn about their own first impression conversation style.

## Chat Personality Types
### ENFJ · 대화 흐름 메이커

채팅에서 자연스럽게 주제를 던지고 이어 붙이며 흐름이 끊기지 않게 합니다. 상대의 말투와 감정을 빠르게 읽고 공감·질문·정리를 섞어 대화를 “굴러가게” 만드는 편이라 단체 채팅에서도 존재감이 큽니다.

### ENTJ · 결론 먼저 치는 타입

대화의 목적과 핵심을 빠르게 잡고, 불필요한 왕복을 줄이려 합니다. “그래서 결론은?”, “다음은 뭐 하면 돼?”처럼 액션 중심 메시지가 많고, 채팅을 의사결정 도구처럼 활용하는 경향이 강합니다.

### ENFP · 주제 점프 채터

한 대화 안에서 아이디어와 감정이 연쇄적으로 튀며 주제가 자주 확장됩니다. 링크, 예시, 비유를 곁들여 말이 풍부해지고, 상대가 흥미를 느끼는 포인트를 발견하면 분위기를 크게 끌어올립니다.

### ENTP · 반문 던지는 채터

상대 말에 “근데 다른 관점은?” 같은 반문을 던지며 논점을 확장합니다. 토론처럼 주고받는 걸 즐기고, 채팅에서 가설·대안·예외를 끌어와 사고를 흔드는 편이라 대화가 깊어지기 쉽습니다.

## ESFJ · 리액션 담당

맞장구와 공감 반응이 빠르고 다양해 상대가 편하게 말하게 합니다. 이모지, 짧은 리액션, 칭찬을 적절히 섞어 분위기를 따뜻하게 만들고, 대화가 어색해지면 먼저 연결 질문으로 흐름을 살립니다.

### ESTJ · 채팅 정리러

대화 내용을 깔끔하게 정리해 혼선이 없게 합니다. “요약하면 1) 2) 3)”처럼 구조화된 메시지가 많고, 일정·역할·우선순위를 빠르게 잡아 단체 채팅에서 진행자 역할을 자주 맡습니다.

### ESFP · 감탄 연발형

반응이 즉각적이고 표현이 풍부해 채팅방의 에너지를 올립니다. 감탄사, 웃음, 리액션을 잘 쓰고, 재미있는 포인트를 빠르게 잡아 대화를 가볍고 즐겁게 유지하는 데 강한 타입입니다.

### ESTP · 즉답형 채터

메시지를 보자마자 바로 반응하며 속도감 있는 대화를 선호합니다. 길게 설명하기보다는 핵심만 툭 던지고, 필요한 정보나 해결책을 빠르게 제시하는 편이라 실시간 상담/문제 해결 채팅에 강합니다.

### INFJ · 한 줄 핵심러

대화를 많이 주도하지는 않지만, 흐름을 오래 지켜보다가 결정적인 순간에 핵심을 짚는 메시지를 보냅니다. 감정의 맥락을 잘 읽어 불필요한 말 없이도 상대에게 “이해받고 있다”는 느낌을 주는 채팅 스타일입니다.

### INTJ · 구조 짜서 치는 타입

채팅 내용을 머릿속에서 정리한 뒤 메시지를 보냅니다. 말의 순서와 논리가 분명하며, 문제 상황에서는 원인–해결–결론 구조로 답장을 구성하는 경향이 강해 설명이 명확합니다.

### INFP · 감정 담아 쓰는 타입

메시지 하나에도 감정과 가치관이 자연스럽게 스며듭니다. 표현은 조심스럽지만 진정성이 느껴지고, 상대의 말에 개인적인 공감이나 경험을 덧붙여 대화를 깊게 만드는 편입니다.

### INTP · 설명 길어지는 타입

질문을 받으면 개념부터 원리까지 차근차근 풀어 설명합니다. 채팅이 점점 장문이 되는 경우가 많고, 정의·비교·예외까지 다루며 논리적으로 정리된 답변을 선호합니다.

### ISFJ · 기억하고 챙기는 타입

이전에 나눈 대화를 잘 기억하고 다시 언급합니다. 상대의 상황이나 감정을 세심하게 챙기며, 필요할 때 조용히 도움이나 확인 메시지를 보내는 안정적인 채팅 스타일입니다.

### ISTJ · 정보만 치는 타입

감정 표현은 최소화하고 사실과 정보 위주로 메시지를 보냅니다. 일정, 수치, 조건을 정확히 전달하며, 애매한 표현보다 명확하고 검증 가능한 내용을 선호합니다.

### ISFP · 조용한 공감 타입

대화에서 크게 튀지는 않지만, 필요한 순간에 따뜻한 공감 한마디를 건넵니다. 말수는 적어도 감정의 결을 잘 맞추며, 부드럽고 차분한 채팅 분위기를 만듭니다.

### ISTP · 최소 메시지형

불필요한 설명을 줄이고 핵심만 전달합니다. 단답형이 많고 메시지 길이는 짧지만, 문제 해결 상황에서는 정확한 포인트를 짚어 효율적인 소통을 하는 타입입니다.

## Personality Index

### 1. 평균 메시지 길이 (Message Length)
한 번의 메시지에서 사용자가 평균적으로 작성하는 글자 수입니다.  
장문 위주의 채팅인지, 핵심만 전달하는 단문형 채팅인지를 판단할 수 있습니다.

- **높음**: 설명형·분석형 (INTP, INTJ, ENFP)  
- **낮음**: 즉답형·요약형 (ISTP, ESTP, ISTJ)

---

### 2. 응답 속도 (Response Time)
상대 메시지를 받은 뒤 첫 답장을 보내기까지 걸리는 시간입니다.  
즉흥적인 반응인지, 숙고 후 응답하는지를 구분하는 데 사용됩니다.

- **빠름**: 즉흥·외향형 (ESTP, ESFP, ESFJ)  
- **느림**: 숙고·내향형 (INTJ, INFJ, INFP)

---

### 3. 질문 비율 (Question Ratio)
전체 메시지 중 질문 형태(?, 의문문)가 차지하는 비율입니다.  
대화를 이끄는 방식이 질문 중심인지, 전달 중심인지 판단합니다.

- **높음**: 탐색·확장형 (ENTP, ENFJ, INFJ)  
- **낮음**: 전달·정리형 (ISTJ, ESTJ)

---

### 4. 감정 표현 밀도 (Emotion Density)
이모지, 감탄사, 감정 단어 사용 빈도를 종합한 지표입니다.  
채팅에서 감정과 리액션이 얼마나 드러나는지를 나타냅니다.

- **높음**: 감정·리액션형 (ESFP, ESFJ, INFP)  
- **낮음**: 정보·논리형 (INTP, ISTJ, INTJ)

---

### 5. 메시지 구조화 지수 (Structure Score)
메시지가 얼마나 정리된 형태로 작성되는지를 측정합니다.  
번호, 줄바꿈, 요약 표현 등의 사용 빈도를 기반으로 계산합니다.

- **높음**: 판단형(J) 성향 (ESTJ, ENTJ, INTJ)  
- **낮음**: 인식형(P) 성향 (ENFP, ISFP, ESTP)

---

## 활용 예시
평균 메시지 길이가 길고, 질문 비율이 높으며,  
감정 표현은 적고 구조화 지수가 높을 경우  
→ **INTJ / INTP 계열 유형 확률이 상승**
